# Absolute Cinema - Frontend

React aplikasi web untuk sistem pemesanan tiket bioskop dengan 4 role interface.

## 🚀 Quick Start

```bash
# Install & Run
npm install
npm run dev
```

## 👥 User Roles

### Customer (/)
- Browse film (Now Playing, Coming Soon)
- Pilih jadwal & kursi
- Booking & payment
- Riwayat pesanan

### Admin (/admin)
- Kelola film (CRUD)
- Kelola jadwal & harga
- Kelola customer & kasir
- Setting kursi

### Owner (/owner)
- Laporan keuangan
- Analytics bisnis

### Cashier (/cashier)
- Booking offline
- Print tiket
- Proses pembayaran online
